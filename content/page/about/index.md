__1. Web 前端__

- 学生时代在蓝色理想 JavaScript 区当版主
- [上海图书馆](https://www.library.sh.cn) 实习录入员，五笔输入繁体近代文章约每分钟 65 个汉字
- 小创业公司写 PHP、ColdFusion、JavaScript

__2. iOS 开发__
